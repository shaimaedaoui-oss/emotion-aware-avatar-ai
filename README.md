# Emotion-Aware 3D Conversational Avatar

## Overview

This project is an AI-based 3D avatar capable of interacting with users through voice and text while adapting its facial expressions based on detected emotions.

The system analyzes user input, detects emotional tone, and generates intelligent responses using an AI model, which are then expressed through both voice and dynamic facial animations.

## Features

* Voice and text interaction
* Speech-to-text processing
* Emotion detection from user input
* AI-generated responses using an LLM API
* Real-time facial expression adaptation

## Technologies

* Python
* NLP (Natural Language Processing)
* Speech-to-Text
* LLM API
* Computer Vision / 3D Animation
